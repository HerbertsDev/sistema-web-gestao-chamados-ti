# Sistema Web de Gestão de Chamados e Incidentes de TI

Projeto front-end desenvolvido para simular um ambiente de gestão operacional de TI, com foco em abertura de chamados, acompanhamento de incidentes, exibição de indicadores e visualização de dados em dashboard.

## Objetivo
Criar uma aplicação web capaz de organizar e apresentar informações operacionais de TI de forma visual, prática e acessível.

## Funcionalidades
- Tela de login
- Dashboard com KPIs
- Cadastro de chamados
- Cadastro de ativos
- Tabelas dinâmicas
- Cards informativos
- Gráficos operacionais
- Persistência de dados com LocalStorage

## Tecnologias utilizadas
- HTML5
- CSS3
- JavaScript
- LocalStorage

## Diferenciais do projeto
- Interface voltada para gestão operacional
- Estrutura visual moderna
- Organização de dados sem back-end
- Simulação prática de fluxo real de suporte e infraestrutura

## Estrutura do projeto
- `login.html`
- `index.html`
- `chamados.html`
- `ativos.html`
- `usuarios.html`
- `relatorios.html`
- `style.css`
- `script.js`

## Como executar
1. Baixe ou clone o projeto
2. Abra o arquivo `login.html` ou `index.html`
3. Utilize o sistema no navegador
